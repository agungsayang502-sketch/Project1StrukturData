# Tugas Array

## Penjelasan
Program ini digunakan untuk mengolah data nilai mahasiswa menggunakan array.

- Input 10 nilai
- Menampilkan nilai tertinggi & terendah
- Menghitung rata-rata
- Menghitung jumlah lulus
